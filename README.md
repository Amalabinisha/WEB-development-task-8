# Simple Blog Layout with Bootstrap 5

Responsive blog page built using Bootstrap 5 CDN.

## Features

- Navbar with brand and links
- Blog cards using Bootstrap grid
- Each card: image, title, description, "Read More" button
- Responsive layout on all screen sizes
- Footer with social icons

## How to Use

1. Copy `index.html` file to your project folder.
2. Open `index.html` in your browser.
3. Edit card content, images, and links as needed for your blog.

## Requirements

- Internet connection (uses Bootstrap CDN)
- Any modern browser

## Customization

- Add more cards for blog posts.
- Change colors, layout, and icons using Bootstrap utility classes.
